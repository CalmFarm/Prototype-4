  일반적인 컴파일 오류 진단 및 수정
  Unity의 스크립팅 API와 관련된 컴파일 오류 진단 및 수정

기초적인 코드 최적화
  코딩 베스트 프랙티스를 올바르게 실행하여 코드 효율성 극대화
  성능 문제 디버그

  인터페이스 사용을 비롯한 추상화 및 캡슐화 원칙을 올바르게 구현하여 코드의 안전성과 사용성 향상
  객체 지향 프로그래밍의 원칙을 올바르게 구현하여 효율적이고 체계적이며 이해하기 쉬운 코드 작성
